## [0.1.0] - 2026-02-14

### Added

- Map-based case reporting with PostGIS geospatial indexing
- Automated jurisdiction assignment using municipal boundary polygons
- Multi-step report submission with client-side image compression
- Case detail screen with photo gallery and real-time timeline
- Moderator review queue with verify/reject/flag actions
- Community flagging with auto-archive at threshold
- Push notifications for case status updates
- Email escalation to government authorities via Resend
- Auto-escalation reminders at 5, 15, and 30 days
- Government portal with folio assignment and official responses
- Inbound email parsing for government replies
- Mercado Pago donation integration (OXXO, SPEI, card)
- Public impact dashboard with resolution metrics
- About page and legal documents (LFPDP-compliant privacy notice)
- Bilingual support (Spanish primary, English secondary)
- Row Level Security policies for jurisdiction-scoped access control
- Design token system with terra cotta/sage color palette
- Web support via react-native-web and mapbox-gl
